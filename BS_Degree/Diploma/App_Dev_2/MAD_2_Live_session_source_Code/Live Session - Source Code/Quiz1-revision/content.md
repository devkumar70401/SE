## Topics to focus in first 4 weeks

### week 1

- JS basics - primitives - (undefined, null, NaN) and datatypes, 
- variables and their declaration (let, const, var)
- scoping of variables
- DOM Api --> document
- Objects and properties

### week 2

- DOM manipulation
- Object and Arrays
- array methods - Map, filter, reduce, find
- function/object methods - apply, call, bind 
- Object api
- Iterations and destructuring
- prototype
- Functions - scoping, normal, arrow, classes (OOP), objects (OOP), getter and setter methods, class inheritance, this keyword 

### week 3

- Vue Basics - Vue instanceof, Vue cdn
- vue directives - v-bind, v-on, v-if, v-for, v-show 
- Options object, properties of options object, data, computed, methods, watchers 
- vue instance nesting

### week 4

- Components - uses and creation 
- registering components - local, global 
- data in components
- props
- emit
